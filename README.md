## 📌 Descrição

Este repositório contém a implementação de um sistema de locação de karts desenvolvido em Portugol, com foco na aplicação de conceitos fundamentais de lógica de programação.

O projeto simula o gerenciamento de uma locadora, permitindo cadastrar karts, realizar locações e devoluções, consultar disponibilidade e calcular o lucro diário, por meio de um menu interativo executado no console.

---

## 🛠 Recursos Utilizados

Linguagem: Portugol
IDE: Portugol Studio
Paradigma: Programação Estruturada
Controle de versão: Git
Plataforma de hospedagem: GitHub

---

## 📋 Estrutura do Projeto

O projeto foi organizado seguindo os seguintes critérios:

Menu interativo com estrutura de repetição enquanto.

Cadastramento de karts com número identificador e valor de locação.

Listagem de karts disponíveis e locados.

Processo de locação e devolução de karts.

Cálculo do kart mais alugado.

Cálculo de lucro diário com base nas locações.

Opção de locação de circuito adicional.

Atualização diária das locações.
